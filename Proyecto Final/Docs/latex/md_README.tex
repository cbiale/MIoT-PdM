Se desarrolla un sistema que lee datos de un sensor de temperatura y humedad ficticio. El sistema tiene dos estados donde\+:
\begin{DoxyItemize}
\item En el primer estado los datos obtenidos se muestran en un lcd y por la UART.
\item En el segundo estado se deja la pantalla fija del lcd y se muestran datos por la UART.
