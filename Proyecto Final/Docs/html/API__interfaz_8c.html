Muestra los datos según las opciones especificadas.

Esta función muestra los datos en dispositivos como LCD y UART, según las opciones proporcionadas. También permite forzar una nueva lectura si se especifica.

Parámetros
mostrar_lcdIndica si se deben mostrar los datos en un LCD.
mostrar_uartIndica si se deben mostrar los datos a través de UART.
forzar_lecturaIndica si se debe forzar una nueva lectura antes de mostrar los datos.
